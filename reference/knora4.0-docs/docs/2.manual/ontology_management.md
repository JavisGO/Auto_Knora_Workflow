# 本体管理

本体管理是 {{aio.name}} 平台的核心基础模块，用于构建和维护企业业务知识图谱。通过定义实体（Entity）、关系（Relation）、事件（Event）与行为（Action），将企业业务对象的语义结构固化为可计算、可复用的知识底座，为后续的智能体推理、数据分析与流程自动化提供稳定的语义支撑。

进入本体管理模块后，左侧导航提供以下子模块入口：

- **语义图谱**：以可视化方式呈现知识本体结构与实体关系
- **业务对象（Entity）**：管理知识图谱中的核心实体类型
- **业务关联（Relation）**：定义实体与实体之间的语义关系
- **事实明细（Event）**：记录业务对象的状态变化与行为轨迹
- **业务操作与行为（Action）**：将业务操作标准化为可调用接口
- **本体自动构建**：基于 AI 自动识别实体、属性与关系
- **流程活动（Logic / Workflow）**：低代码编排智能体工作流
- **自主推理（Knora Claw）**：Agentic 自主规划引擎
- **工具与模型**：管理可调用的外部工具与大模型资源
