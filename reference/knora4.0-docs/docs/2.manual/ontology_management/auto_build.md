# AIGO 本体自动构建

AIGO 本体自动构建功能基于 AI 能力，支持从业务数据、数据表结构或业务文档中自动识别实体、属性与关系，并生成本体草稿，大幅降低本体建模的门槛与工作量。

在左侧导航中点击**AIGO 本体自动构建**，进入自动构建引导页。

## 3 实体自动识别

系统将基于输入数据自动识别候选实体，并在列表中展示识别结果：
![实体自动识别结果](picture/auto_build_entity.png){ width="100%", loading=lazy }
/// caption

用户可对自动构建的实体、关系、事件进行二次确认和编辑，确认或修改实体的名称、业务描述，完成全部核查后，再执行本体发布，确保结构的准确性与完整性。
