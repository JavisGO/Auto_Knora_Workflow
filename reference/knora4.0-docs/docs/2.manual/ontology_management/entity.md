# 业务对象（Entity）管理

业务对象（Entity）是知识图谱中对业务核心要素的定义。通过创建实体类型，可将企业业务中的核心对象（如"客户"、"合同"、"设备"等）以统一的语义结构进行建模，为后续的数据接入、关系构建与智能分析提供基础。

在左侧导航中点击**业务对象（Entity）**，进入实体管理列表页。

## 1 新增实体

点击页面右上角的**新建实体**按钮，进入实体创建向导，共分为四个步骤：

**第一步：基本信息**

![新增实体 - 基本信息](picture/entity_create_step1.png){ width="100%", loading=lazy }
/// caption
图2-3 新增实体 - 基本信息
///

填写以下信息：

- **实体名称**：实体类型的中文名称，用于在平台界面展示（必填）。
- **实体标识符（API Name）**：实体在系统内的唯一英文标识，用于接口调用（必填，创建后不可修改）。
- **实体描述**：对该实体的业务含义进行说明，帮助团队成员理解其语义（选填）。
- **图标**：为实体选择一个图标，在图谱中直观区分不同实体类型（选填）。

填写完成后，点击**下一步**。

**第二步：数据映射**

![新增实体 - 数据映射](picture/entity_create_step2.png){ width="100%", loading=lazy }
/// caption
图2-4 新增实体 - 数据映射
///

配置该实体的数据来源，将实体与已接入的数据源表进行绑定：

- 选择**数据源**和对应的**数据表**，系统将自动识别表中的字段。
- 指定**唯一标识字段**，该字段将作为实体实例的唯一键，用于去重与关联。

填写完成后，点击**下一步**。

**第三步：属性定义**

![新增实体 - 属性定义](picture/entity_create_step3.png){ width="100%", loading=lazy }
/// caption
图2-5 新增实体 - 属性定义
///

为实体定义属性列表，每个属性需配置：

- **属性名称**：属性的中文名称（必填）。
- **属性标识符**：属性的英文唯一标识（必填）。
- **数据类型**：文本、数值、时间、布尔、枚举等类型（必填）。
- **是否必填**：标记该属性在数据接入时是否为必填项（选填）。

点击**添加属性**可继续新增，填写完成后点击**下一步**。

**第四步：属性映射**

![新增实体 - 属性映射](picture/entity_create_step4.png){ width="100%", loading=lazy }
/// caption
图2-6 新增实体 - 属性映射
///

将第三步定义的属性与第二步选择的数据源字段进行对应关系配置：

- 从左侧属性列表中选择实体属性，在右侧下拉框中选择对应的数据源字段。
- 支持一个属性映射到多个字段（字段合并）或配置固定值。

完成映射后，点击**保存**，实体定义创建完毕。

## 2 实体查看与编辑

在实体列表中，点击某个实体名称，可进入该实体的详情页，查看完整的定义信息，包括基本信息、属性列表及数据映射配置。

点击详情页右上角的**编辑**按钮，可对已创建的实体进行修改，包括：

- 更新实体名称、描述与图标。
- 调整属性结构（新增、编辑、删除属性）。
- 重新配置数据映射关系。

!!! note "注意"
    实体标识符（API Name）在创建后**不可修改**，请在创建时谨慎填写。

## 3 实体标签管理

标签用于对实体进行分类管理，便于在大量实体定义中快速筛选。

在实体列表中，选中目标实体，点击操作列中的**标签**图标，可在弹窗中为该实体添加、编辑或删除标签。

- 一个实体可添加多个标签。
- 标签支持在列表页顶部的筛选栏中使用，快速过滤特定分类的实体。

## 4 实体检索

实体列表页顶部提供检索功能：

- **关键词搜索**：在搜索框中输入实体名称关键词，实时过滤列表。
- **标签筛选**：点击标签筛选下拉框，选择一个或多个标签，筛选特定分类的实体。

## 5 实体发布

完成实体定义后，需将其发布才能使语义结构与属性配置对后续流程生效。

在实体列表或详情页中，点击**发布**按钮，系统将对该实体的定义进行校验，校验通过后正式发布上线。

## 6 实体授权

支持对特定实体进行用户授权，控制不同用户对该实体的操作权限。

在实体详情页，点击**授权管理**，在弹窗中进行配置：

- **权限类型**：分为**管理**（可编辑、发布、授权）和**查看**（仅可查看定义与数据）两种权限级别。
- **授权用户**：选择需要授权的平台用户或用户组。
- **授权时长**：可设置授权的有效期限，到期后自动失效；也可设置为**永久**授权。
