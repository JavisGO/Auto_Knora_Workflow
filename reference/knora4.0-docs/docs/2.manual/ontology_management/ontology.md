# 语义图谱

语义图谱是以可视化方式呈现知识本体结构与实体关系的核心模块。用户可在图谱画布上直观探查实体、关系与事件之间的语义连接，清晰掌握业务对象的关联脉络与知识结构全貌，为本体的构建与维护提供直观的操作视角。

点击左侧导航中的**语义图谱**，进入图谱画布界面。

## 1 实体搜索

在画布上方的搜索框中，选择搜索类型为**实体**，输入实体名称、类型或属性关键词，点击搜索按钮，系统将在图谱中高亮定位到匹配的实体节点。

- 支持按名称、类型、属性等多维度条件组合检索。
- 检索结果会在画布中以高亮形式展示，点击节点可查看实体详情。

## 2 关系搜索

在搜索框中选择搜索类型为**关系**，输入关系名称或关联实体关键词，可查询实体间的结构关系，直观呈现业务对象之间的语义连接路径。

## 3 事件搜索

在搜索框中选择搜索类型为**事件**，输入事件名称或主体实体关键词，可查询特定主体的业务行为与状态变化记录，从时序和动态视角检索图谱中的事实明细数据。

## 4 实体查看

在图谱画布中**点击实体节点**，右侧面板将展开该实体的详情，包括：

- 实体类型与唯一标识
- 属性定义列表
- 与其他实体的关联关系

## 5 关系查看

在图谱画布中**点击关系连线**，右侧面板将展开该关系的详情，包括：

- 关系类型与方向（来源实体 → 目标实体）
- 关系属性定义
- 关联的实体信息

## 6 画布调整

画布底部提供全局操作按钮：

- **缩放按钮**：对画布进行放大或缩小（或使用鼠标滚轮），也可选择"适应画布"使所有节点按比例填满屏幕。
- **全屏按钮**：将画布切换为全屏模式，获得更大的操作空间。
- **平移**：在画布空白处按住鼠标左键拖动，可整体平移图谱视角。
- **布局调整**：对节点布局进行手动调整，拖动节点至合适位置后释放即可。
